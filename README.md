By leveraging advanced artificial intelligence, Readynx offers personalized career roadmaps, real-time resume analysis, and interactive mock interviews to help students and professionals become job-ready.

The platform features a stunning **Cyber Aurora** theme, providing a futuristic, immersive, and premium user experience with neon aesthetics and deep space backgrounds.

## рџ“¦ Installation & Setup

Follow these steps to get Readynx running on your local machine.

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Steps

1.  **Clone the Repository**

    ```bash
    git clone https://github.com/your-username/readynx.git
    cd readynx/client-site
    ```

2.  **Install Dependencies**

    ```bash
    npm install
    ```

3.  **Configure Environment Variables**
    Create a `.env` file in the root directory and add the following:

    ```env
    VITE_API_URL=http://localhost:5000/api
    VITE_GOOGLE_CLIENT_ID=your_google_client_id
    VITE_GEMINI_API_KEY=your_gemini_api_key
    ```

    _(Note: Replace placeholders with your actual API keys/URLs)_

4.  **Run Development Server**
    ```bash
    npm run dev
    ```
    The application will launch at `http://localhost:5173`.
